第04回 git challenge (2016.11.20)

過去最高の熾烈な優勝争いとなった第4回大会が2016/11/20(日)に無事終了。 2016年入社エンジニアと2017年新卒内定者によるシステムの改修を行い、自動採点システムの高速化を実現した今回は、参加者の解答スピードも速く、現場エンジニア社員も驚くほどでした。 終盤まで参加者全員が高い集中力を切らさず、走り抜けてくれた結果、大会史上初の3チーム同率1位優勝で幕を閉じました。
